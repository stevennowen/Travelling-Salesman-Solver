# TSP-IF2211

### How to Run
``` bash
ruby tsp_solver.rb
```
### Input 
![image](https://github.com/user-attachments/assets/c19de8ef-7132-4f07-a40a-a5bb57202e78)

2. Masukan edge untuk setiap kota (input dipisah dengan spasi)
